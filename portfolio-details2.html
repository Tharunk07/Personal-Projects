Project Description

Project information

  • Title:Leaves Disease Classification using CNN
  • Tech Stacks:CNN,Tensorflow,Django
  • Project Github URL: Github Link

Developed an intelligent system leveraging Convolutional Neural Networks to accurately classify various leaf diseases across plant species. Trained on a comprehensive dataset, it analyzes leaf images and identifies diseases, enabling early detection and timely interventions. User-friendly interface allows seamless image upload and disease diagnosis. Scalable and adaptable to incorporate new diseases and plant varieties.
